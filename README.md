# Perception

**Perception** is a script engine built for Minecraft 1.8.9 

> **Note:** Support for later versions of Minecraft is planned for the future once the 1.8.9 version is stable or sm


### Workspace & Setup

**Setting up workspace:**
1. Clone the repository
2. Specify your project SDK to **Java 8** (it might not work with other versions).

**Building:**
Run the `mvn clean package` command. Once the process is complete the built `.jar` will be in the `/target` directory.

**Running in IDE:**
To launch the client in your IDE RUN `Start.java` and **specify the working directory as `./test_run/`** if you want
All saves, resource packs, etc., will be there.
